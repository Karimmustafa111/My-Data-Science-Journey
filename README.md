### 2. üö¢ Titanic Survival Analysis (Real-World Data)
**File:** `titanic_eda.py`

This project marks my transition to **real-world datasets**. Using the famous Titanic dataset, I performed a comprehensive **Exploratory Data Analysis (EDA)** to determine the factors that influenced survival rates.

**Key Challenges Solved:**
* **Data Cleaning:** Handled missing values in `Age` (Imputation) and `Embarked`, and dropped the empty `Deck` column.
* **Insights:**
    * **Gender:** Female passengers had a significantly higher survival rate.
    * **Class:** First-class passengers were prioritized during rescue operations.

**Key Visualizations:**
* Survival Count (Countplot).
* Survival by Gender & Class (Barplots).
* Age Distribution (Histplot).
